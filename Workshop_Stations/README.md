# Workshop Stations
The Wearable Workshop is divided into four stations that introduce participants to the core skills required for wearable prototyping. Each station includes a guided tutorial, a sample project, and support from workshop supervisors.

By completing all four stations, student will gain experience in soft fabrication, electronics assembly, programming, and mechanical design before applying these skills to an independent project.

## Station 2 – EVA Foam
In this station, you will create a lightweight EVA foam glasses case with an adjustable hanging strap that can be attached to a backpack or bag.

Going through this tutorial, you will learn how to:

- Create a paper template
- Transfer a pattern onto EVA foam
- Cut EVA foam safely and accurately
- Heat form EVA foam into a three-dimensional shape
- Assemble parts using hot glue
- Attach and adjust Velcro fasteners
